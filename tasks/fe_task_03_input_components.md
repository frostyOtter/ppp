# Frontend Task 03: Input Components (Upload & Settings)

## Context
You are part of the **Frontend Team**. You are responsible for the "Input" section of the UI, where users provide the file and configuration.

## Objectives
1.  Build a `FileUpload` component.
2.  Build a `ParserSettings` component.

## Detailed Instructions
1.  **`FileUpload` Component:**
    *   Create `src/components/FileUpload.tsx`.
    *   Implement a drag-and-drop zone.
    *   Visual feedback on drag over.
    *   Display selected filename.
    *   Accept only `.pdf` files.
2.  **`ParserSettings` Component:**
    *   Create `src/components/ParserSettings.tsx`.
    *   Create a Dropdown (Select) for `ParserType`.
    *   (Optional) Create number inputs for `start_page` and `max_pages`.
3.  **Layout:**
    *   Arrange these components nicely (e.g., Sidebar or Top Bar) using TailwindCSS classes.

## Definition of Done
*   [ ] User can drop a PDF file or click to select one.
*   [ ] User can choose a parser from a list.
*   [ ] Components look professional (styled with Tailwind).
